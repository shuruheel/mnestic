# mnestic (engine core)

This crate (directory `cozo-core`, published as **mnestic**) is the engine core of **mnestic**, an independently maintained fork of CozoDB. It is **not** official CozoDB and is not affiliated with or endorsed by its original authors. Original design © Ziyang Hu and the Cozo Project Authors.
